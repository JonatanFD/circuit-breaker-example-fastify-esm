# Redis Fastify

## Docker
```shell
docker build -t redis-fastify .
```

## Redis

```shell
docker run -d -p 6379:6379 --name my-redis redis
```

## Docker Compose

To start the application with Docker Compose, run:

```shell
docker-compose up 
```

To stop the application with Docker Compose, run:

```shell
docker-compose down
```
